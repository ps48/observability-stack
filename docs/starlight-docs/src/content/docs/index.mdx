## Welcome to OpenSearch - Observability Stack

OpenSearch Observability Stack is an open-source, OpenTelemetry-native platform for full-stack observability. Monitor your services, infrastructure, and AI agents from a single pane of glass.

## What You Can Do

### Send Data
Instrument your applications and infrastructure with OpenTelemetry. Collect traces, logs, and metrics from any language, framework, or platform.

### Investigate
Query, explore, and correlate your observability data. Drill into traces, search logs, and analyze metrics to find root causes fast.

### Application Monitoring
Visualize service dependencies, track latency and error rates, and monitor application performance across your distributed systems.

### Dashboards & Visualize
Build custom dashboards with real-time panels, charts, and heatmaps. Query metrics with PromQL and trace data with PPL.

### Logs
Ingest, search, and correlate logs with traces. Full-text search with structured field queries using PPL.

### Metrics
Prometheus-compatible metrics ingestion with PromQL support. Custom dashboards and RED metrics computed from traces.

### Traces
Distributed tracing with span-level detail. Trace context propagation, service maps, and trace analytics.

### AI Observability
Trace AI agent workflows with OpenTelemetry GenAI semantic conventions. Visualize agent execution graphs, monitor token usage, and debug agent behavior with Python and JavaScript SDKs.
